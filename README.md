# Análise de Dados de E-commerce com SQL

## Objetivo

Este projeto foi desenvolvido para responder a questões de negócio através da análise de dados de uma empresa fictícia de e-commerce utilizando SQL.

## Skills Demonstradas

* SQL
* INNER JOIN
* LEFT JOIN
* GROUP BY
* HAVING
* Subqueries
* CTEs
* Business Analysis
* Data Modeling

## Base de Dados

O projeto utiliza quatro tabelas:

* Clientes
* Produtos
* Pedidos
* Categorias

## Perguntas de Negócio Respondidas

1. Quais foram os produtos comprados por cada cliente?
2. Que clientes nunca fizeram compras?
3. Quantos pedidos cada cliente realizou?
4. Qual é o produto mais caro da loja?
5. Qual é o preço médio dos produtos?
6. Quais os produtos com preço acima da média?
7. Quais os clientes que compraram produtos acima da média de preço?
8. Qual cliente gastou mais dinheiro?
9. Quanto cada cliente gastou no total?
10. Quais clientes gastaram acima da média dos clientes?
11. Qual categoria vendeu mais produtos?
12. Qual categoria gerou mais faturação?

## Ferramentas Utilizadas

* SQLite
* SQL
* SQLite Online

## Principais Insights

* Identificação dos clientes com maior valor para o negócio.
* Identificação das categorias mais rentáveis.
* Deteção de clientes sem histórico de compras.
* Comparação de produtos e categorias através de métricas agregadas.
